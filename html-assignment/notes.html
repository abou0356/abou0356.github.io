HTML tags

mark:
used to define text that should be marked or highlighted for reference or notation purposes.
br:
inserts a single line break.
div:
defines a division or section in an HTML document and acts as a generic container for other HTML elements.
p:
is a fundamental element used to define a paragraph of text on a web page.
blockquote:
defines a section of text that is an extended quotation from another source.
